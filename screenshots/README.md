\# AWS CloudWatch Monitoring Project



\## Overview

This project demonstrates EC2 monitoring using AWS CloudWatch with dashboards, alarms, and SNS notifications.



\## Services Used

\- Amazon EC2

\- Amazon CloudWatch

\- Amazon SNS



\## Features

\- CPU utilization monitoring

\- CloudWatch dashboard creation

\- Alarm setup for CPU threshold (e.g., 80%)

\- Email notifications via SNS



\## Architecture

EC2 → CloudWatch Metrics → Alarm → SNS → Email Notification
